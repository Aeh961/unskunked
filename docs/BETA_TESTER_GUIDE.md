# Unskunked Beta Tester Guide

Thank you for testing Unskunked. The goal of this beta is to learn whether the app helps beginner anglers plan better trips and understand what to try next.

## How To Install Or Run

For local development:

```bash
npm install
npm run ios
npm run android
```

For Expo Go:

```bash
npm start
```

Then scan the Expo QR code or open the project in an iOS Simulator or Android Emulator.

## What To Test

- Can you finish onboarding without confusion?
- Does the Home screen make the next step obvious?
- Can you find a beginner-friendly waterbody?
- Can you plan a fishing trip in under two minutes?
- Do fish, rig, and regulation summaries feel clear?
- Can you log a trip and understand your stats?
- Can you report feedback and export beta data?

## Beginner Test Scenarios

1. You have never fished before and only own a spinning rod.
2. You only have worms and want to catch something near Seattle.
3. You want to fish Green Lake in July.
4. You got skunked and want to know what to change.
5. You want to learn one knot and one rig before leaving home.

## Fishing Trip Planning Scenarios

- Plan a trout trip at Green Lake.
- Plan a perch trip at Lake Washington.
- Plan a dock or pier session.
- Change experience from Beginner to Advanced and see if the plan still makes sense.
- Save a plan, start a trip, then log the result.

## How To Report Bugs

Use the in-app Feedback screen.

Choose the closest type:

- Bug report
- Feature request
- Confusing regulation
- Wrong fish recommendation
- Wrong waterbody info
- General feedback

Include the screen name, what happened, and what you expected.

## Known Limitations

- Regulation data is mocked for beta planning.
- Washington is the only region with full mock data.
- Oregon, Idaho, and California are placeholders.
- There is no backend or account sync yet.
- Weather, tides, stocking reports, and GPS are not connected yet.
- Screenshots and exports use local device data.

## Safety And Regulation Reminder

Unskunked is for planning and education only. Do not rely on this app as legal advice. Always verify official rules, emergency rules, licenses, seasons, limits, size rules, closures, and gear restrictions with the official fish and wildlife agency before fishing or keeping fish.
