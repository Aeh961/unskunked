# Privacy

Unskunked is local-first for public beta.

## Location

Location is used only to sort nearby waters and shellfish locations on the device. If permission is denied, the app uses manual Washington fallback locations.

## Trip Storage

Trip logs, saved plans, notes, and local fishing statistics are stored on the device. They are not uploaded.

## Favorites

Favorite fish, rigs, knots, waters, and shellfish locations are stored locally.

## Offline Storage

Offline pack selections, cached condition snapshots, source metadata, demo data, and recent searches are stored locally.

## Analytics

The beta uses local-only beta events for product insight screens. No hidden third-party tracking is included.

## Feedback

Feedback is stored locally until the user chooses to export or share it.

## Future Cloud Plans

Future cloud sync should be opt-in, explain exactly what is uploaded, and include export/delete controls before launch.
